# DNSCHIE ABRASIVE B2B Website

This is a static single-page B2B website for DNSCHIE ABRASIVE.

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select **main** branch and **/root** folder.
6. Save and wait for GitHub Pages to publish.

## File Structure

```text
/
├── index.html
├── README.md
└── .nojekyll
```

The website is packaged as a single HTML file. All CSS, JavaScript and website content are included in `index.html`.
